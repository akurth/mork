Chapter 2. Setup

Prerequisites. Mork needs Java 6 or higher. JRE is fine, you don't need the JDK.To checkout and run the example, you need Maven 3 and Subversion installed. Note that the example have no scripts for windows, you have to manually launch them.

Install. There's no Mork distribution to download or install. Mork is published to Maven central, you'll just add a dependency to your project (as described later).
